# Breaking Apps with Passmark AI

AI-powered stress testing and regression testing project using Passmark AI and Playwright.

## 🚀 About This Project

This project was created for the Breaking Apps Hackathon.

The goal is to:
- Test real-world applications using AI
- Simulate user interactions automatically
- Detect UI issues and broken workflows
- Generate reproducible regression tests

Built using:
- Passmark AI
- Playwright
- TypeScript
- Node.js

## ⚡ Installation

Clone repository:

```bash
git clone <your-repo-url>
cd Passmark-calcom
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run Test

Run all Playwright tests:

```bash
npx playwright test
```

Run AI Cal.com test:

```bash
npx playwright test ai-breaking-calcom/tests/ai-calcom.spec.ts
```
